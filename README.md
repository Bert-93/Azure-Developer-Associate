# Azure Developer Associate

Cree soluciones de un extremo a otro en Microsoft Azure para crear Azure Functions, implementar y administrar aplicaciones web, desarrollar soluciones que usen Azure Storage, etc.

## Contenido

### [01 Implementación de aplicaciones web de Azure App Service](./01%20Implementaci%C3%B3n%20de%20aplicaciones%20web%20de%20Azure%20App%20Service)

Explora cómo implementar, configurar y escalar aplicaciones web en Azure App Service, incluyendo ranuras de implementación para despliegues sin tiempo de inactividad.

#### [.NET — API y Web Frontend](./01%20Implementaci%C3%B3n%20de%20aplicaciones%20web%20de%20Azure%20App%20Service/.NET)

Contiene una solución .NET con un backend API REST (ImagesController) y una aplicación web frontal (Index.cshtml) para galería de imágenes integrada con Azure Blob Storage.

---

### [02 Implementar Azure Functions](./02%20Implementar%20Azure%20Functions)

Aprende a desarrollar, depurar y desplegar Azure Functions serverless con diferentes triggers y bindings para automatizar tareas y procesos.

#### [.NET — Http2 y Http3](./02%20Implementar%20Azure%20Functions/.NET)

Ejemplo de dos funciones HTTP que registran datos en Azure Table Storage: una usando bindings de salida de Functions y otra con acceso manual a la tabla.

---

### [03 Desarrollo de soluciones que usan Blob Storage](./03%20Desarrollo%20de%20soluciones%20que%20usan%20Blob%20Storage)

Guía sobre Azure Blob Storage: exploración, gestión del ciclo de vida y operaciones comunes (Az CLI, AzCopy, SAS, SDKs).

---

### [04 Desarrollo de soluciones que usan Azure Cosmos DB](./04%20Desarrollo%20de%20soluciones%20que%20usan%20Azure%20Cosmos%20DB)

Introducción a Azure Cosmos DB: exploración, modelos de facturación (RU), APIs soportadas, coherencia y ejemplos server-side.
